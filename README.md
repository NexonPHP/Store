# Nexon

Nexon is a free and open-source Minecraft webstore that integrates directly with your game server. It makes managing stores, subscriptions, and automated delivery simple, and works across multiple servers and proxies.

### Features

* Deep Minecraft integration: automatically grant ranks, run commands, and sync player profiles.

* Multi-server support: works with multiple servers, proxies, and game types.

* Automated delivery: orders are processed and delivered instantly.

* Subscription management: handle recurring purchases and renewals with ease.

* Custom plugins and extensions: add your own features if you want.

* Centralized data: keep all player and transaction information in one place.

* Open source and self-hosted: you own your data, and there are no platform fees.
